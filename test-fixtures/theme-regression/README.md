# Theme Regression Fixture

Open these files in the Extension Development Host when comparing Fleetty themes
against Fleet:

- `sample.py`
- `sample.kt`
- `sample.json`
- `sample.md`
- `privacy-policy.html`

Use `Developer: Inspect Editor Tokens and Scopes` on representative tokens such
as Python `if`, Kotlin `sealed class`, JSON keys, Markdown headings, and HTML/CSS
tokens like `--bg`, `#0f1117`, `@media`, `:root`, `body`, `font-family`, and
`<meta>`.
